# Sound Assets

This directory contains sound effects for the vocabulary app.

## Required Sound Files

Place the following MP3 files in this directory:

1. **correct.mp3** - Pleasant chime sound for correct answers
   - Duration: ~0.5-1s
   - Volume: Moderate
   - Suggested sources: freesound.org, zapsplat.com
   - Search terms: "success chime", "correct answer", "positive notification"

2. **wrong.mp3** - Gentle buzz sound for wrong answers
   - Duration: ~0.5-1s
   - Volume: Moderate
   - Suggested sources: freesound.org, zapsplat.com
   - Search terms: "error buzz", "wrong answer", "negative notification"

3. **complete.mp3** - Celebration sound for quiz completion
   - Duration: ~1-2s
   - Volume: Moderate
   - Suggested sources: freesound.org, zapsplat.com
   - Search terms: "success fanfare", "achievement", "celebration"

## Requirements

- Format: MP3
- File size: < 100KB each
- Royalty-free/licensed for commercial use
- Quality: 44.1kHz, 128kbps minimum
